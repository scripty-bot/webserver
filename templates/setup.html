Getting Started

To get started, ensure you have a channel dedicated to Scripty's output.
Scripty can spam a lot when there's a lot of activity in VC, so it's best to keep it in a separate channel.
What it's named is up to you. For this guide, we'll call it #transcriptions.

Step 1

Head into the channel you made for Scripty's output, and type /setup.
This slash command has a few required arguments, and a few optional ones.
You will immediately be prompted to select a channel to output to.

Select the channel you made for Scripty's output.
Optionally, you can select the server's language, and whether Scripty should be verbose.
These default to English and off, respectively.
If you would like to see more about these options, continue to Step 1a. Otherwise, submit the slash command and continue to Step 2.

Step 1a

You can optionally set Scripty's language and verbosity.

The language is useful if your server does not speak English.
Scripty will only listen for voice in the language you select, so it should be the language your server, or most members, use in voice chats.
Currently, Scripty only supports a small number of languages, but more are planned.


The verbosity option is useful only if you want to see more of what Scripty is doing.
We recommend leaving it off, as it can be quite spammy.

Once you've selected your options, submit the slash command and continue to Step 2.

Step 2

You will be prompted to agree to Scripty's privacy policy and terms of service.
You can also find a link to both below.


Privacy Policy

Terms of Service

If you agree, click the Agree button and continue to Step 3.
If you disagree, click the Disagree button. This will halt the setup process.

Step 3

This completes the setup process.
You can now make Scripty join a voice channel and start transcribing.
To do so, you must first join a voice channel, then type /join in any channel.


Hint: the /join command accepts an optional argument, a voice channel to join. If you don't specify one, Scripty will join the channel you're in.

Step 4

Now try speaking something into your microphone.
Given up to 10 seconds, Scripty should transcribe what you said.
Keep in mind that Scripty is continually evolving, so it very likely will not be perfect.
That's okay! It's still learning.

Step 5

If you'd like to stop Scripty from transcribing, you can type /leave in any channel.


Note

To prevent abuse of our service, we limit both how long you can use Scripty in a single channel, and how many users can use Scripty in a single channel.
By default, these limits are 30 minutes and 5 users, respectively.
If you reach the 30 minute limit, Scripty will automatically leave the channel. You can make it join again by typing /join in any channel.
If you reach the 5 user limit, Scripty will simply ignore any additional users.
These limits may seem low, but keep in mind Scripty is completely free. We can't afford to run it for hours on end.
If you would like to increase these limits, check out our Premium plans.
We massively appreciate your support!
